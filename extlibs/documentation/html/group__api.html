Detailed Description

The NetWeaver RFC C API allows custom C programs to.

  • query the R/3 DDIC for function module and structure definitions
  • call ABAP function modules in R/3
  • receive ABAP function calls from R/3 and
  • create/read/write/traverse the data containers necessary for the above.
